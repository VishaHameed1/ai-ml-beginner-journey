# 🚀 AI & ML Beginner Journey

A comprehensive roadmap from Python fundamentals to Machine Learning models.

## 📂 Project Structure

### 🟢 Level 1: Python Foundations
* **Number Guessing Game**: Logic, loops, and random modules.
* **Simple Calculator**: Functional programming and error handling.
* **Password Generator**: String manipulation and randomization.
* **Student Record System**: Data persistence using JSON and Dictionaries.

### 🟡 Level 2: Data Science Entry
* **Netflix Analysis**: Exploratory Data Analysis (EDA) using Pandas.
* **Salary Analysis**: Data visualization with Matplotlib and GroupBy operations.

### 🔴 Level 3: Machine Learning
* **House Price Prediction**: Linear Regression for continuous value prediction.
* **Spam Classifier**: Logistic Regression for text classification.
* **Customer Segmentation**: Unsupervised learning using K-Means Clustering.
