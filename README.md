# 🚀 AI & ML Beginner Journey: From Logic to Vision AI
A 13-project roadmap documenting my transition from Python fundamentals to advanced Multimodal AI.

## 🛠️ Tech Stack & Skills
* **Languages:** Python 3.14
* **AI Models:** Gemini 2.5 Flash (LLM & Vision), Logistic Regression, Linear Regression, K-Means.
* **Data Science:** Pandas, Matplotlib, NumPy, Yahoo Finance API.
* **DevOps/Tools:** Git, GitHub (Version Control), API Integration, Environment Variables (.env).

---

## 📂 Project Roadmap

### 🟢 Level 1: Python Foundations (Logic & Data Structures)
* **Student Record System**: Implemented a "mini-database" using JSON persistence.
* **Logic Suite**: Built a Password Generator, Calculator, and Guessing Game to master loops and error handling.

### 🟡 Level 2: Data Science (Analysis & Visualization)
* **Netflix & Salary Analysis**: Used Pandas for Exploratory Data Analysis (EDA) and Matplotlib to visualize global data trends.

### 🔴 Level 3: Machine Learning (Predictive Modeling)
* **House Price Prediction**: Mastered **Linear Regression** for continuous data.
* **Spam Classifier**: Implemented **Logistic Regression** and Vectorization to categorize text.
* **Customer Segmentation**: Used **K-Means Clustering** for unsupervised pattern recognition.

### 🔵 Level 4: Real-World Integration (APIs & Live Data)
* **Weather App**: Integrated OpenWeatherMap API with secure `.env` key management.
* **Stock Visualizer**: Built a live financial tracker using `yfinance` and time-series plotting.

### 🟣 Level 5: Advanced AI (LLMs & Computer Vision)
* **Personal AI Chatbot**: Leveraged **Gemini 2.5 Flash** for a context-aware conversational agent.
* **Vision AI**: Implemented Multimodal AI that analyzes and explains visual content (e.g., Identifying cat behaviors like the Flehmen response).

---
## 📈 Key Learnings
* **API Security**: Learned to protect sensitive keys using `.gitignore` and environment variables.
* **Version Control**: Mastered Git `rebase` and conflict resolution in a collaborative workflow.
* **Multimodal AI**: Moved beyond text-only models to systems that "see" and "reason."
